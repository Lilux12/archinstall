ВАРИАНТ 1: Самый быстрый способ
─────────────────────────────────
1. Скопируйте папку архива на USB флешку
2. Загрузитесь с Live ISO Arch Linux
3. Смонтируйте флешку: mount /dev/sdX1 /mnt
4. Зайдите в папку: cd /mnt/archinstalll2.0
5. Запустите скрипт: sudo bash start.sh

   Это сделает автоматически:
   ✓ Установит русский шрифт (cyr-sun16)
   ✓ Обновит pacman базу
   ✓ Установит Python 3
   ✓ Установит все зависимости
   ✓ Проверит требуемые утилиты
   ✓ Запустит установщик


ВАРИАНТ 2: Ручной способ
────────────────────────
1. Загрузитесь с Live ISO
2. Подключитесь к интернету
3. Скопируйте содержимое установщика
4. Выполните команды вручную (см. ниже)


ВАРИАНТ 3: Загрузка из интернета
────────────────────────────────
# Загрузить скрипт прямо в памяти
curl -L https://raw.githubusercontent.com/.../start.sh | sudo bash

РУЧНЫЕ КОМАНДЫ (если не работает start.sh)
═══════════════════════════════════════════════════════════════════════

# 1. Установить русский шрифт
setfont cyr-sun16

# 2. Обновить pacman
pacman -Sy --noconfirm

# 3. Установить Python
pacman -S --noconfirm python python-pip

# 4. Установить зависимости
pip install pythondialog psutil requests pyyaml

# 5. Установить необходимые утилиты
pacman -S --noconfirm parted arch-install-scripts

# 6. Перейти в папку установщика
cd /path/to/archinstalll2.0

# 7. Запустить установщик
# Русский интерфейс:
sudo python3 main.py --lang ru

# Английский интерфейс:
sudo python3 main.py --lang en

РЕШЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════

Проблема: "Permission denied" при запуске start.sh
─────────────────────────────────────────────────────
Решение: chmod +x start.sh && sudo bash start.sh

Проблема: "Python not found"
───────────────────────────
Решение: pacman -S python python-pip

Проблема: "Dialog not found"
────────────────────────────
Решение: pip install pythondialog
       или: pacman -S dialog

Проблема: Нет интернета
────────────────────────
Решение: 
  - Проверить кабель Ethernet
  - Использовать WiFi: iwctl (встроено в Live ISO)
  - Проверить DNS: ping 8.8.8.8

Проблема: Установка зависает
──────────────────────────────
Решение:
  - Проверить интернет соединение
  - Посмотреть логи: tail -50 /var/log/archinstall.log
  - Перезагрузить скрипт

ПОДДЕРЖКА И ПОМОЩЬ
═══════════════════════════════════════════════════════════════════════

Если что-то не работает:

1. Проверьте логи
   tail /var/log/archinstall.log

2. Читайте EXAMPLES.md в папке архива

3. Проверьте интернет
   ping archlinux.org

4. Убедитесь что запускаете от root
   sudo bash start.sh

5. Смотрите Arch Wiki
   https://wiki.archlinux.org
